# Base Converter

A simple web application that converts numbers between different bases:  
Binary (2), Octal (8), Decimal (10), and Hexadecimal (16).

This project was created as part of an introductory web development exercise and focuses on basic JavaScript logic, input validation, and DOM manipulation.

---

## Features

- Select a source base (FROM)
- Select a target base (TO)
- Validate input according to the selected base
- Convert numbers between bases using JavaScript
- Display the result clearly on the page
- Simple and clean user interface

---

## Technologies Used

- HTML  
- CSS  
- JavaScript  

---

## Project Structure

```
index.html    → Page structure and UI  
style.css     → Styling and layout  
main.js       → Base conversion logic  
```

---

## How to Run

1. Download or clone the repository.  
2. Open `index.html` in any modern browser.  
3. No installation or server is required.
